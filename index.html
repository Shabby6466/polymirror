🔒 Live Trading Wallet Required

Live mode places real on-chain orders on Polymarket. Paste the private key of a Polygon wallet that holds USDC. It's encrypted before being stored and never sent back to the browser.

📘 How to get a private key (click to expand)

Recommended: create a fresh wallet just for this bot. Don't reuse your main MetaMask key.

Option A — New MetaMask account (easiest):

  1. Open MetaMask → click the account icon → Add account or hardware wallet → Add a new account.
  2. Switch to the new account, switch network to Polygon.
  3. Click the three-dot menu → Account details → Show private key (enter password).
  4. Copy the key (starts with 0x…) and paste below.

Option B — Generate one (most secure):

  1. Visit vanity-eth.tk (or use cast wallet new) to generate a key locally.
  2. Save the address + private key somewhere safe.

Then fund the wallet's EOA on Polygon:

  1. Bridge or buy USDC.e on Polygon to the wallet address — this is what the bot will spend on trades.
  2. Send a small amount of MATIC (≈ $1) for gas.
  3. We will prompt you to Approve USDC/CTF in the next step (one-time, costs gas).

⚠ The funds must be on the wallet's EOA address directly. If you use the Polymarket web app, your USDC is in a separate proxy wallet the bot can't see — that key won't work.

✅ Key Connected

Before trading, the Polymarket Exchange Smart Contract needs one-time permission to spend your USDC and CTF (shares). This setup requires a small amount of MATIC for gas.
